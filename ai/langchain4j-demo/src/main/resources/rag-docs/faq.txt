蜗牛学苑常见问题 FAQ
====================

一、报名与入学
------------
问：报名需要什么条件？
答：大部分课程零基础可学，大数据课程要求有 Java 基础。年满 18 周岁，高中及以上学历即可报名。

问：如何报名？
答：可通过以下方式报名：
1. 官方网站在线报名
2. 拨打校区电话预约报名
3. 到校区现场咨询报名
4. 联系招生老师微信报名
报名需提供身份证复印件、一寸照片 2 张，缴纳定金后保留名额。

问：可以免费试听吗？
答：可以的！所有校区均提供一周免费试听，试听满意后再报名缴费。

问：开课时间是怎么安排的？
答：全日制班每月 1 日、15 日开新班；周末班每周末上课。具体时间以各校区公告为准。

二、学费与缴费
------------
问：学费可以分期吗？
答：支持 3/6/12 期免息分期，需提供本人身份证和银行卡信息。

问：如果中途退学，学费怎么算？
答：开课 7 天内退学全额退款；开课 1 个月内按比例退款；超过 1 个月原则上不退款。

问：除了学费还有其他费用吗？
答：学费已包含所有课程资料费、教材费、实训平台使用费。住宿费和生活费需自理。

三、学习与教学
------------
问：上课时间是怎样的？
答：全日制班周一至周五 9:00-12:00 上课，14:00-17:00 上机练习，18:00-20:00 晚自习辅导；周末班周六周日 9:00-17:00 上课。

问：请假制度是怎样的？
答：学员需提前一天向班主任请假，紧急情况可电话/微信请假。每月不超过 3 次请假，超出部分需补课。

问：学完能达到什么水平？
答：Java 全栈课程毕业后可独立开发企业级 Web 应用；Python 课程毕业后可独立完成数据采集、分析和 Web 开发；前端课程毕业后可独立完成 PC 端和移动端页面开发；大数据课程毕业后可独立搭建和维护大数据平台。

四、就业服务
----------
问：包就业吗？
答：蜗牛学苑提供终身就业推荐服务，毕业即享受就业指导、简历优化、模拟面试、企业推荐等全方位就业服务。2024 年整体就业率 95.3%，但这不承诺"包就业"，最终取决于个人能力和努力。

问：合作企业有哪些？
答：与 2000+ 家企业建立人才合作关系，其中包括：华为、腾讯、阿里巴巴、字节跳动、京东、美团等一线互联网企业，以及大量中小型科技公司和创业公司。

问：就业的平均薪资是多少？
答：根据不同城市和课程，平均起薪范围如下：
- 一线城市（北上广深）：10K-18K
- 新一线城市（成都、杭州、武汉等）：7K-13K
- 二线城市：5K-9K

五、其他
--------
问：可以免费重学吗？
答：当期课程没学会的，可以免费跟着下一期班重学，不限次数，直到学会为止。

问：有线上课程吗？
答：有！蜗牛学苑在线平台提供部分课程的录播版本，但线上课程不包含项目实战和就业服务。建议有条件的学生选择面授课程。
