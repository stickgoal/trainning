一、Java 全栈开发课程
------------------
课程级别：零基础入门 → 中级 → 高级
学制：6个月（全日制）/ 10个月（周末班）
适合人群：零基础转行者、计算机专业毕业生、在职提升者
授课方式：线下面授 + 录播回放 + 项目实战

课程大纲：
1. Java 基础语法（4周）：数据类型、流程控制、面向对象、集合框架、泛型、异常处理
2. 数据库技术（3周）：MySQL 核心、JDBC、MyBatis、MyBatis-Plus
3. Web 开发（4周）：HTML/CSS/JavaScript、Servlet/JSP、Vue.js 前端框架
4. 框架技术（6周）：Spring、Spring MVC、Spring Boot、Spring Cloud 微服务
5. 项目实战（4周）：电商系统全栈开发（包含订单、支付、库存等核心模块）

学费：¥22,800
就业方向：Java 开发工程师、后端开发工程师、全栈工程师
平均起薪：8K-15K（根据城市不同）
