# 商品文案写作技能

## 何时使用此技能
- 用户需要为商品撰写营销文案
- 用户提供了商品名称、特点、目标人群等信息
- 需要生成吸引人的推广文案

## 使用流程
1. 确认商品信息：名称、特点、卖点、目标人群
2. 分析目标人群的痛点和需求
3. 生成 3 个不同风格的文案备选
4. 每个文案控制在 50 字以内
5. 标注每个文案的适用场景（朋友圈、电商详情页、短视频口播等）

## 文案风格要求
- 简洁有力，突出核心卖点
- 使用口语化表达，贴近消费者
- 包含行动号召（如"立即下单""限时优惠"）
- 避免夸大宣传，保持真实可信

## 输出格式
```
【商品】{商品名称}
【卖点】{核心卖点总结}

文案1（适用场景：朋友圈）
{文案内容}

文案2（适用场景：电商详情页）
{文案内容}

文案3（适用场景：短视频口播）
{文案内容}
```
