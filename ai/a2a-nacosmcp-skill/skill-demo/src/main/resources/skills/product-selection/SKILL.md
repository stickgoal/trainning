# 选品分析技能

## 何时使用此技能
- 用户需要分析市场趋势，选择适合的商品品类
- 用户提供了目标人群、预算范围、市场方向等信息
- 需要给出有数据支撑的选品建议

## 使用流程
1. 明确目标人群画像（年龄、性别、消费能力、兴趣爱好）
2. 分析当前市场热点和趋势
3. 列出 3-5 个推荐品类
4. 对每个品类进行 SWOT 分析（优势、劣势、机会、威胁）
5. 给出最终选品建议和优先级排序

## 分析维度
- **市场规模**：该品类的市场规模和增长趋势
- **竞争程度**：头部品牌占比、新进入者难度
- **利润空间**：毛利率、客单价区间
- **用户需求**：目标人群的购买意愿和频次
- **供应链**：货源稳定性、物流成本

## 输出格式
```
【目标人群】{人群画像}

【推荐品类 TOP 3】

1. {品类名称}
   - 市场规模：{描述}
   - 竞争程度：{描述}
   - 利润空间：{描述}
   - 推荐指数：⭐⭐⭐⭐⭐

2. {品类名称}
   ...

【最终建议】{一句话总结}
```
